# Resident Evil Requiem Autosplitter/Load Remover
Autosplits and load removal for Resident Evil VRequiem

Support: https://ko-fi.com/nikoheart

# Instructions
Right-click Livesplit and select Edit Splits and change the game name to "Resident Evil Requiem".
Once the game name has been inputted, you should see Autostart, Autosplit and Autoreset by Nikoheart.
Click the "Activate" button.
Configure what settings you would like to split on by clicking "Settings" and checking of specific check boxes in each of the grouped sections,
Afterwards, create your splits/segments with any name that you'd like to input, and create the correct amount of splits for the amount of boxes you have checked within the groups.
Also make sure to ensure that your main timer is showing the Game Timer - you can do this by right-clicking Livesplit -> Compare Against -> Game Time.
